## 功能介绍

程序目前分为以下两个模式：

- 单机模式：暂未开放。
- 联机模式：当前主要可用版本（有时候连不上服务器就是崩了，可以私信踢我一脚）。

使用流程如下：

1. 在主界面输入玩家昵称。
2. 选择对应模式并进入。
3. 进入大厅后，可以寻找其他玩家创建的房间。
4. 输入房间密码后加入房间。
5. 也可以在底部输入密码，自行创建房间。
6. 双方玩家都进入房间并准备后，即可开启 BP。
7. BP 流程与游戏内武斗赛流程基本保持一致。

## 注意事项

1. 当前版本的单机版暂未开放。
2. BP 环节暂未完成 BO3 机制。
3. 当前对应版本为 **2025 年 11 月武斗赛比赛服**。（该版本已经过期，如果有人能提供新版忍者选择页面截图，后续可以进行更新。）
4. 部分忍者立绘暂未更新。
5. BP 完成后的对策评分功能暂未开放。

## 制作背景

制作本程序的起因是看到一些主播和玩家想要模拟比赛场景，但缺少一个专门用于 BP 环节的工具。为了让模拟比赛流程更加完整、清晰，也方便双方进行角色禁选和选择，因此开发了这个 BP 模拟程序。

同时，本程序也是一次关于联网操作的学习与研究。如果有想要学习相关内容的朋友，可以加入 QQ 群交流、提问或获取源文件。
